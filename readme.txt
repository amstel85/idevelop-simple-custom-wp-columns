A WordPress plugin to easily add, remove, and customize columns in admin list tables for posts, pages, and custom post types.

== Description ==

Simple Custom WP Columns provides an intuitive, code-free way for WordPress users to customize their admin list tables. Display relevant data like post meta or taxonomy terms directly in your post, page, and custom post type lists, improving content management efficiency.

== Installation ==

1. Upload the `idevelop-simple-custom-wp-columns` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Settings > Simple Custom WP Columns to configure your custom columns.

== Frequently Asked Questions ==

= Can I add columns for custom fields (post meta)? =
Yes, the plugin allows you to select post meta as a data source for your new columns.

= Can I display taxonomy terms in columns? =
Absolutely, you can choose any registered taxonomy to display its terms in a column.

= Is it compatible with custom post types? =
Yes, the plugin fully supports all registered custom post types.
